# Personal Portfolio Website

Hi there 👋 — this repository contains the source code for my personal portfolio website.  
The project’s primary goal is to create a clean, effective, and performant portfolio site while showcasing my ability to write concise, efficient, and optimized code.

---

## 🎯 Project Goals

- Build a portfolio that highlights my work clearly and accessibly.
- Showcase **code quality, optimization practices, and effective design decisions**.
- Iterate based on **user feedback** to improve usability and clarity.

---

## 🛠️ Tech Stack (MVP)

- **Design**: [Figma] — design tokens, wireframes.
- **Framework**: [Astro](https://astro.build/) + [TailwindCSS](https://tailwindcss.com/) — lightweight static-first approach with utility-first styling.
- **Images**: Stored in `/public/images` for simplicity.
- **Version Control**: Git + GitHub — version history and collaboration-ready structure.
- **Testing**:
  - [Vitest](https://vitest.dev/) for unit/component validation.
  - [Playwright](https://playwright.dev/) for end-to-end browser testing.
  - Accessibility and performance checks (axe + Lighthouse).

---

## 📐 Development Approach

- **Concise Code Practices**

  - Favor clarity and simplicity over over-engineering.
  - Reusable, composable components with minimal abstraction.

- **Optimization Practices**

  - Astro’s static rendering for speed and SEO.
  - Lightweight Tailwind setup with design tokens.
  - Optimized images and lazy loading.

- **User Feedback Integration**
  - Prioritize content clarity and navigation flow.
  - Collect feedback on readability, accessibility, and usability to guide iteration.
